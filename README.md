# CICD on Azure VM

Install Jenkins on Ubuntu VM, then install `nvm`, and LTS version of nodejs.

```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source .bashrc
nvm install --lts
```
